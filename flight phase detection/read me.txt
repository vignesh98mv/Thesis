Repository Contents

Predictions_program/
Includes all Python scripts used to develop and execute the final phase prediction pipeline.

AI_prompts_and_responses.pdf
Provides examples of the major AI prompts and corresponding responses used during the development of this thesis.

labelled_training_data.csv
Contains the labelled training dataset used to fine-tune the BERT-base-uncased model for flight phase classification.

final_phase_results.ods
Contains the final flight phase prediction results generated by the system. This file includes:

The predicted output
The expected (ground truth) output
The calculated average deviation between predicted and expected flight phases
